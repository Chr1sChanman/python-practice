# Commands
## To clone a repo into a folder that already has Git initialized

`rm -rf path/to/cloned-folder/.git`
> Remove nested .git

`git rm -f --cached path/to/cloned-folder`
> Remove embedded repo reference from idx

`git add path/to/cloned-folder`
> Add the folder and follow normal process from here
